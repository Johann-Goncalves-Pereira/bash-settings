## SetUp

Clone in your home directory.

Fist make the symbolic link.

```zsh
ln -s .shell-config/zsh/.zshrc .zshrc
ln -s .shell-config/zsh/.bashrc .bashrc
ln -s .shell-config/zsh/.p10k.zsh .p10k.zsh
```
